# 🛒 E-commerce Sales Analysis & Recommendation System

## 📌 Project Overview
This project focuses on analyzing e-commerce transactional data to extract meaningful business insights and build a simple product recommendation system. It combines data analytics and basic machine learning techniques to simulate real-world business scenarios.

## 🎯 Objectives
- Analyze sales performance and customer behavior  
- Identify top-selling products and key markets  
- Track sales trends over time  
- Build a recommendation system to suggest related products  
- Create an interactive dashboard for business insights  

## 🛠️ Tools & Technologies
- Python (Pandas, Matplotlib)
- Streamlit (for mini application)
- Power BI (for dashboard)
- Apriori Algorithm (Association Rule Mining)

## 📊 Data Processing
- Cleaned raw dataset by removing null and invalid values  
- Converted data types (date, numeric fields)  
- Filtered relevant data for better performance  

## 📈 Key Analysis Performed
- Top 10 products by sales  
- Monthly sales trend analysis  
- Country-wise sales distribution

## 🤖 Recommendation System
- Built using **Apriori algorithm**  
- Identifies frequently bought product combinations  
- Suggests related products based on customer purchase patterns  

## 🌐 Streamlit Application
- Developed a simple web app for product recommendations  
- Users can select a product and get suggested items  
- Demonstrates practical implementation of recommendation logic  

## 📊 Power BI Dashboard
- Interactive dashboard with key KPIs:
  - Total Sales  
  - Total Orders  
  - Total Customers
  - Total Product Sold 
- Visualizations include:
  - Sales trends over time  
  - Top-performing products  
  - Geographic sales distribution  
- Added filters for dynamic analysis  

## 💡 Key Insights
- A small number of products contribute significantly to total revenue  
- Sales show variation across different months  
- Certain countries generate higher sales compared to others  
- Frequently purchased products can be leveraged for cross-selling
